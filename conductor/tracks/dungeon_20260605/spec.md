# Specification: Masmorra Exploration

## Overview
Implement the Masmorra Exploration feature, allowing users to "check in" at real-world gyms to earn Recompensas. The UI will be built according to the Stitch design specifications.

## Functional Requirements
- Integrate Geolocator for GPS location.
- Integrate Google Places API to validate gym locations.
- Implement Check-in logic.
- Create Masmorra UI utilizing the Stitch MCP for styling.

## Acceptance Criteria
- App successfully remissoes location permissions.
- Gym locations are accurately identified.
- UI components reflect Stitch design guidelines.

## Out of Scope
- Complex Masmorra minigames.
